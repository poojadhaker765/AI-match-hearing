# TalentThread - AI Powered Job Board Platform

TalentThread is a modern job board frontend platform that connects candidates and employers.

## Features

### Candidate
- Role-based login
- Candidate dashboard
- Job search interface
- Application tracking UI
- Resume upload interface
- Skill matching UI

### Employer
- Employer dashboard
- Job posting interface
- Applicant ranking UI
- Hiring pipeline dashboard
- Analytics dashboard

## Tech Stack

Frontend:
- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

Future Backend:
- React.js
- Node.js / Express
- PostgreSQL
- Python NLP Resume Parser
- Machine Learning Matching Algorithm
